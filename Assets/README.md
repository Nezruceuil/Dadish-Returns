# Dadish Returns

It's a Dadish fan game I made because I didn't want to wait
until the next dadish game comes out!

and you can play it for free yey!

## Installation:

https://nezruceuil.itch.io/dadish-returns

## Controls:

# With Keyboard:
arrow keys or A/Q'left' and D'right' and mouse

# With Controller:
joystick or joypad for movements and chose
A for selecting
L and R for world switch
+ for pause

## Features
- Works Offline !
- Supports Controller !
- Save your progress !
