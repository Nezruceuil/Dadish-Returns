# Dadish Returns

Is a Dadish fan game i made i because i didn't wanna wait
until the next dadish game came out !
and you can play it for free yey !

## Installation:

https://nezruceuil.itch.io/dadish-returns

## Controls:

# With Keyboard:
arrow keys or Q'left' and D'right' and mouse
space to jump and start !

# With Controller:
joystick or joypad for movements and chose
A for selecting
L and R for world switch
+ for pause

## Features
- Works Offline !
- Supports Nintendo Controller !
- Save your progress !
